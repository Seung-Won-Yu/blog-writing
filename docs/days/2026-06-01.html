기초상식 · 정처기
#정보보안

웹 애플리케이션에서 사용자 입력값을 검증하지 않아 악의적인 SQL 구문이 실행되는 공격 기법은?

  1. ① DDoS 공격
  2. ② SQL 인젝션
  3. ③ 스니핑(Sniffing)
  4. ④ 무차별 대입 공격
정답 & 해설 보기

정답: ②

SQL 인젝션은 입력값 검증 부족을 악용해 조작된 SQL 구문을 주입하여 인증 우회나 데이터 유출을 일으키는 공격으로, 입력값 검증과 매개변수화 쿼리(Prepared Statement)로 방어한다. DDoS는 가용성 공격, 스니핑은 패킷 도청, 무차별 대입은 비밀번호 추측 공격이다.
