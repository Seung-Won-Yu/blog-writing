여러 디스크에 데이터를 분산 저장(스트라이핑)해 입출력 성능을 높이지만, 패리티가 없어 디스크 하나만 고장 나도 전체 데이터를 잃는 RAID 레벨은?

  1. ①RAID 0
  2. ②RAID 1
  3. ③RAID 5
  4. ④RAID 6

보기를 클릭해 정답을 확인하세요.

정답: ①

RAID 0은 스트라이핑으로 성능·용량 효율이 좋지만 중복성이 없어 한 디스크 장애 시 전체 데이터를 잃는다. RAID 1은 미러링(복제), RAID 5/6은 패리티로 각각 1개·2개 디스크 장애를 견딘다.
