#터미널

터미널이 빨라지는 fzf — 명령·파일을 흐릿하게 검색

긴 명령을 외우거나 ls로 헤매지 말고, fzf로 히스토리·파일을 실시간 퍼지 검색해 골라 쓴다.

  1. 설치 후 셸 연동(Ctrl+R: 명령 히스토리 퍼지 검색, Ctrl+T: 파일 경로 검색)
  2. vim $(fzf) 처럼 다른 명령의 인자로 파이프해 파일을 즉시 선택
  3. git branch | fzf 로 브랜치 목록에서 골라 체크아웃 등 조합 활용

상어는 나무보다 오래된 생물이다

상어의 조상은 약 4억 5천만 년 전부터 바다를 누볐다. 반면 육상에 진짜 '나무'라 할 만한 식물이 등장한 것은 약 3억 5천만~3억 9천만 년 전이다. 즉 상어는 나무는 물론 토성의 고리, 북극성보다도 먼저 지구에 있었던 셈이다.
