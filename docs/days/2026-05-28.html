컴파일러와 인터프리터에 대한 설명으로 옳은 것은?

  1. ①인터프리터는 소스 전체를 한 번에 목적 코드로 번역한다
  2. ②컴파일러는 한 줄씩 해석하여 즉시 실행한다
  3. ③컴파일러는 실행 전 전체를 번역해 목적 파일을 생성한다
  4. ④인터프리터는 실행 속도가 컴파일러보다 항상 빠르다

보기를 클릭해 정답을 확인하세요.

정답: ③

컴파일러는 소스 코드 전체를 실행 전에 한 번에 목적 코드(목적 파일)로 번역하여 이후 실행 속도가 빠르다. 인터프리터는 소스를 한 줄씩 해석·실행하므로 일반적으로 실행 속도가 더 느리지만 개발·디버깅이 편리하다.
