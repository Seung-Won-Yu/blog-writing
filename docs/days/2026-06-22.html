#개발

정규식 lookaround — 매칭하되 '잡아먹지' 않기

앞뒤 조건은 확인만 하고 결과에는 포함하지 않는 lookahead/lookbehind로 치환을 깔끔하게 만든다.

  1. (?=...) 뒤쪽 긍정 전방탐색 / (?<=...) 앞쪽 긍정 후방탐색 — 위치만 매칭
  2. 예: 천 단위 콤마 → \d(?=(\d{3})+$) 위치에 ',' 삽입
  3. 예: 'price:' 뒤 숫자만 → (?<=price:)\d+ 로 숫자만 캡처
  4. 부정형은 (?!...) / (?<!...) — 특정 패턴이 '아닐 때'만 매칭

바나나는 아주 약하게 방사능을 띤다

바나나에는 칼륨이 풍부한데, 그중 극히 일부가 방사성 동위원소인 칼륨-40이라 바나나는 미량의 방사선을 낸다. 이 정도가 워낙 작아 방사선 노출을 비교할 때 '바나나 등가 선량(BED)'이라는 비공식 단위로 쓰기도 한다. 물론 아무리 먹어도 몸이 알아서 칼륨을 조절하므로 해롭지 않다.
