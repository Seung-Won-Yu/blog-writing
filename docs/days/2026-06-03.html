  1. ① 하향식 통합 테스트
  2. ② 상향식 통합 테스트
  3. ③ 빅뱅 통합 테스트
  4. ④ 회귀 테스트
정답 & 해설 보기

정답: ②

상향식 통합 테스트는 최하위 모듈부터 위로 통합하며 상위 모듈을 대신하는 테스트 드라이버(Driver)를 사용한다. 반면 하향식 통합은 상위부터 진행하며 하위 모듈을 대신하는 스텁(Stub)을 사용한다.
