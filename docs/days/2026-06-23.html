#에디터

VS Code 다중 커서 — 반복 편집을 한 번에

같은 단어/패턴을 여러 곳에서 동시에 고칠 때 찾기-바꾸기보다 다중 커서가 빠르고 안전하다.

  1. Ctrl+D: 현재 선택 단어와 동일한 다음 항목을 하나씩 추가 선택
  2. Ctrl+Shift+L: 동일 단어 전체를 한 번에 선택
  3. Alt+클릭: 원하는 위치마다 커서를 직접 추가 / Alt+Shift+드래그: 열(세로) 선택
