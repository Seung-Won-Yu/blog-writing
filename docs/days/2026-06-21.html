#개발

파이썬 프로젝트는 venv + 버전 고정이 기본

전역 설치 대신 프로젝트별 가상환경을 쓰고 의존성 버전을 핀 고정하면 '내 PC에선 됐는데' 문제를 막는다.

  1. python -m venv .venv 로 가상환경 생성 후 활성화(.venv\Scripts\activate / source .venv/bin/activate)
  2. pip install 로 패키지 설치, pip freeze > requirements.txt 로 버전 고정
  3. 협업자는 pip install -r requirements.txt 로 동일 환경 재현
  4. .venv 는 .gitignore 에 넣어 커밋하지 않는다
