분할 정복(Divide and Conquer) 기법을 사용하며 평균 시간 복잡도가 O(n log n)인 정렬 알고리즘은?

  1. ①버블 정렬
  2. ②선택 정렬
  3. ③퀵 정렬
  4. ④삽입 정렬

보기를 클릭해 정답을 확인하세요.

정답: ③

퀵 정렬은 피벗을 기준으로 분할하여 재귀적으로 정렬하는 분할 정복 알고리즘으로 평균 시간 복잡도가 O(n log n)이다(최악은 O(n^2)). 버블·선택·삽입 정렬은 모두 평균 O(n^2)의 시간 복잡도를 갖는다.
