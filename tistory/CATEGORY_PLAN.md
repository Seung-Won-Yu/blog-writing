# 티스토리 카테고리 운영안

블로그 방향: 개발 공부 기록과 AI/IT 데일리 뉴스룸을 함께 보여주는 개발자 포트폴리오형 블로그.

## 추천 카테고리

```text
개발 뉴스룸
  └─ 데일리 IT 뉴스
  └─ AI 트렌드
  └─ 개발 용어 정리

개발 공부
  └─ Java
  └─ Python
  └─ SQLD
  └─ 정보처리기사

프로젝트
  └─ AIVIS 개발일지
  └─ 개인 프로젝트

회고
  └─ 학습 회고
  └─ 프로젝트 회고
```

## 기존 글 이동 기준

- `자바/나만의 정리` -> `개발 공부/Java`
- `PYTHON` -> `개발 공부/Python`
- `SQLD` -> `개발 공부/SQLD`
- `정보처리기사` -> `개발 공부/정보처리기사`
- `회고록` -> `회고/학습 회고`
- `[개발일지 #...] AIVIS` 글 -> `프로젝트/AIVIS 개발일지`
- 매일 자동 생성 글 -> 현재 블로그에 있는 `데일리IT뉴스`

## 자동 생성 글 기본값

- 제목: `[데일리 IT 뉴스] YYYY. M. D - 오늘의 개발 뉴스와 정처기 문제`
- 카테고리: `데일리IT뉴스`
- 태그: `AI, IT뉴스, 개발뉴스, 정처기, 개발용어, 데일리다이제스트`
- 발행 전 확인: 원문 링크 3개가 정상인지, 요약이 너무 길지 않은지, 정처기 문제 답이 이상하지 않은지 확인

## 관리자에서 하는 순서

1. `관리자 > 콘텐츠 > 카테고리 관리`에서 위 구조를 만든다.
2. 기존 글을 한 번에 이동한다.
3. `데일리IT뉴스` 카테고리에 자동 생성 글을 테스트로 2~3개 올린다.
4. `관리자 > 꾸미기 > 메뉴`에서 상단 메뉴를 `홈`, `뉴스룸`, `프로젝트`, `공부`, `태그`로 정리한다.
5. 사이드바는 `카테고리`, `최근글`, `태그`, `전체 방문자`만 남긴다.
